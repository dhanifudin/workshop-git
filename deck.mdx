# Scenario 1

## Git basic

---

## Initialize Repository

<ol>
  <Appear>
    <li>Open your terminal or git bash (in windows)</li>
    <li>Create a directory <code>mkdir hello-git</code></li>
    <li>Initialize repository <code>git init</code></li>
    <li>Check hidden subdirectory called <em>.git</em></li>
  </Appear>
</ol>

---

## Check Repository Status

<ol>
  <Appear>
    <li>Run following command, inside your repository<code>git status</code></li>
    <li>The directory is called <b>Working Directory</b></li>
    <li>All changed files will be <b>tracked</b></li>
    <li>All new files are <b>untracked</b> by default</li>
  </Appear>
</ol>

---

## Add Files


<ol>
  <Appear>
    <li>You need to add files to the staging area</li>
    <li>Git has three areas</li>
      <ol>
        <li>Working Directory</li>
        <li>Staging Area</li>
        <li>Repository</li>
      </ol>
  </Appear>
</ol>

---

## Add Files (continued...)

<ol>
  <Appear>
    <li>Create a new file called <b>hello.md</b></li>
    <li>Edit the content and put your name<code>Hello, I'm dhanifudin. I want to learn Git</code></li>
    <li>Check repository status</li>
    <li>Compare with the previous status output</li>
    <li>Add file with the following command <code>git add hello.md</code></li>
  </Appear>
</ol>

---

## Commiting files

<ol>
  <Appear>
    <li>Move file from staging area into repository</li>
    <li>Add message to log history</li>
    <li>Please run command below<code>git commit -m "Add message hello.md"</code></li>
  </Appear>
</ol>

---

## Ignoring files

<ol>
  <Appear>
    <li>Sometimes we need to ignore files to be untracked.</li>
    <li>Git use pattern in <em>.gitignore</em></li>
  </Appear>
</ol>

---

## Example .gitignore

```
# Ignore all files with extensions .tmp
*.tmp
# Ignore node_modules directory
node_modules/
```

[More Examples](https://github.com/github/gitignore)

Run command `git log` from your working directory.
